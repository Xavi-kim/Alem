Alem (Әлем) — Персональный органайзер для КЗ/СНГ
"Твой мир в одном месте"
Комплексное мобильное приложение, объединяющее отслеживание смен, бюджетирование в тенге, личные каналы, привычки, сроки и многое другое для рынка Казахстана/СНГ.
🎯 Почему Alem?
Ни один конкурент не объединяет все необходимые нам функции:

Отслеживание смен (расписание работы)
Бюджет в тенге (личные финансы в казахстанской валюте)
Задачи и сроки (с приоритетами)
Отслеживание привычек (со счётчиком серий)
Личные каналы (как приватный Telegram)
Модуль фитнеса (тренировки + таймеры отдыха)
Работа в оффлайне (без зависимости от сервера)

Идеально подходит для студентов, фрилансеров и профессионалов Казахстана/СНГ.
🛠️ Технологический стек

Фронтенд: React 18 + Tailwind CSS
База данных: IndexedDB (Dexie.js) — оффлайн-первый подход
Мобильность: Capacitor → APK для Google Play
Маршрутизация: React Router v6
Графики: Recharts
Анимации: Framer Motion
Иконки: Lucide React
Богатый текст: TipTap
Даты: date-fns

📱 Функции

Панель управления — Обзор всех активностей
Смены — Трекер расписания работы
Задачи — Список дел с приоритетами
Сроки — Дедлайны по предметам (ЕНУ/ВТ-33)
Привычки — Трекер привычек со сериями
Бюджет — Личные финансы в ₸
Каналы — Личные заметки и сообщения
Фитнес — Логирование тренировок + таймер отдыха
Заметки — Быстрые заметки
Статистика — Аналитика и инсайты
Настройки — Конфигурация приложения
