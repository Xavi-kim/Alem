# Alem (Әлем) — Персональный организатор для КЗ/СНГ

**"Твой мир в одном месте"**

Комплексное мобильное приложение для отслеживания рабочих смен, управления бюджетом в тенге, личных каналов, привычек, дедлайнов и многого другого на казахском/русском рынке.

## 🎯 Почему Alem?

Ни один конкурент не объединяет все необходимые функции:

- **Трекер смен** — отслеживание рабочего расписания
- **Бюджет в ₸** — управление личными финансами в казахстанских тенге
- **Задачи и дедлайны** — список дел с приоритетами
- **Трекер привычек** — отслеживание привычек со стриками
- **Личные каналы** — как приватный Telegram
- **Фитнес модуль** — логирование тренировок + таймер отдыха
- **Работает оффлайн** — не требует сервера

Идеально подходит для студентов, фрилансеров и профессионалов Казахстана/СНГ.

## 🛠️ Технологический стек

- **Фронтенд**: React 18 + Tailwind CSS
- **База данных**: IndexedDB (Dexie.js) — offline-first подход
- **Мобильность**: Capacitor → APK для Google Play
- **Маршрутизация**: React Router v6
- **Графики**: Recharts
- **Анимации**: Framer Motion
- **Иконки**: Lucide React
- **Богатый текст**: TipTap
- **Работа с датами**: date-fns

## 📱 Функции

1. **Панель управления** — Обзор всех активностей
2. **Смены** — Трекер рабочих смен
3. **Задачи** — Список дел с приоритетами
4. **Дедлайны** — Дедлайны по предметам (ЕНУ/ВТ-33)
5. **Привычки** — Трекер привычек со стриками
6. **Бюджет** — Личные финансы в ₸
7. **Каналы** — Личные заметки и сообщения
8. **Фитнес** — Логирование тренировок + таймер отдыха
9. **Заметки** — Быстрые заметки
10. **Статистика** — Аналитика и инсайты
11. **Настройки** — Конфигурация приложения
